DenHa框架 其主要特性包括：

 + 路由支持[PSR-4]
 + 项目无层级限制 
 + 接口化开发 提供详细调试信息
 + 配置和路由目录独立
 + 提供便利助手函数
 + 提供MYSQL类库查询

> DenHa框架的运行环境要求PHP5.6以上。
